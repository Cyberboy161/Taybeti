k Taybeti – Secure Offline Notes

🔐 Taybeti

Taybeti means personal or private in Kurdish (Sorani) and this app will achieve it using the most advanced techniques listed below — first of its kind and completely free. Everyone can see the code how it works on the GitHub page.

⬇ Download APK View on GitHub →

🛡️ Self‑Verifying Login

No password hash stored. Login is verified by decrypting a known string — nothing to crack.

🔑 Per‑Note Keys

Each note has its own passphrase and independent AES‑256‑GCM encryption. No shared secret.

🕵️ Covert Sharing

Disguise encrypted messages as normal‑looking YouTube/Instagram links. Nobody suspects a thing.

📵 100% Offline

No internet permission. No cloud, no telemetry, no backdoors. Your data never leaves the device.

🔒 Custom Keyboard

Built‑in keyboard prevents system keyloggers and predictive text from leaking your secrets.

📖 Open Source (MIT)

Every line of code is public. Auditable, forkable, and free forever.
