Taybeti Notesharing

Total privacy. Total security. Total you.

Taybeti means private, personal, and exclusive in Kurdish — and that's exactly what this app delivers. Built from the ground up for those who refuse to compromise on privacy. Every note is encrypted with its own key. No cloud, no tracking, no backdoors. Open source. Free forever. View the code on GitHub.

🔐 Per‑Note Encryption

Each note has its own passphrase with independent AES‑256‑GCM encryption. No master key. No shared secrets.

🛡️ Self‑Verifying Login

No password hash stored anywhere. Login works by decrypting a known string — nothing exists to crack.

📵 100% Offline

Zero internet permission. No cloud sync. No telemetry. No analytics. Your data never leaves the device.

⌨️ Custom Secure Keyboard

Built-in keyboard blocks system keyloggers and predictive text. Type without leaving traces.

🎨 Rich Drawing & Shapes

Freehand drawing panel with multiple brushes, shapes, color wheel, and eraser. Draw, sketch, annotate securely.

🖼️ Image Layers & Templates

Place images in front of or behind text. Choose page templates — lined, grid, dotted, Cornell. All encrypted.

🕵️ Covert Sharing

Disguise encrypted notes as normal YouTube or Instagram links. Nobody suspects anything.

📖 Open Source (MIT)

Every line of code is public. Auditable, forkable, and free forever. Trust through transparency.
