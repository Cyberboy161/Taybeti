📖 What Does Open Source Mean?

And why it's the reason you can trust Taybeti with your secrets.

🔍 Open Source — explained simply

Open source means the source code — the actual instructions that make the app work — is publicly available for anyone to read, inspect, and verify. Think of it like a restaurant where the kitchen is made of glass: you can see exactly how your food is prepared. With closed‑source apps, the kitchen is behind a concrete wall — you have no idea what's happening inside.

🐙 What is GitHub?

GitHub is the world's largest platform for hosting open source code. It's where developers share their projects so the whole world can see, review, and contribute. Every change to Taybeti's code is publicly tracked on GitHub — who made it, when, and why. Nothing is hidden. View Taybeti on GitHub →

🤝 Why should you trust it?

1. Anyone can audit it

Security researchers, privacy activists, and curious developers can all read the code and verify that it does exactly what we say — and nothing else. No hidden backdoors survive public scrutiny.

2. It can't secretly change

Every single code change is publicly logged on GitHub with a timestamp and author. A company can't quietly add spyware in an update — the community would spot it instantly.

3. You're not locked in

If Taybeti ever disappears, anyone can take the code and continue the project. Your data format is documented and open — no vendor lock‑in. Your notes are yours forever.

4. MIT License = real freedom

Taybeti is released under the MIT license — one of the most permissive open source licenses. Anyone can use, modify, and distribute the code for any purpose, even commercially. There are genuinely no strings attached.

🔐 Why this matters for your privacy

When an app is closed‑source, you're taking the developer's word that it's secure. With Taybeti, you don't need to trust us — you can verify. Independent experts can confirm that there's no backdoor, no telemetry, and no cloud leak. That's what real trust looks like: not a promise, but proof.
